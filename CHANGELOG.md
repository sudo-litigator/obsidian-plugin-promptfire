# Changelog

All notable changes to `Promptfire` will be documented in this file.

## 0.1.0 - 2026-04-01

- Initial public release of `Promptfire` for Obsidian.
- Added profile-based context compilation for vault conventions, current notes, related notes, and working rules.
- Added configurable source definitions for active notes, files, folders, outgoing links, backlinks, and search queries.
- Added source extractors for full note, frontmatter only, body only, heading-filtered sections, and code-block-only collection.
- Added deterministic budgeting with per-source limits, per-block limits, and global output limits.
- Added multiple output targets including clipboard, new note, append to note, append to active note, scratchpad note, and deep links.
- Added multiple export formats: Markdown, XML, and JSON.
- Added interactive preview controls for enabling sources, reordering sources, toggling blocks, recompiling output, and saving snapshot profiles.
- Added vault-native configuration support through `.promptfire.json`.
