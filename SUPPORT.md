# Support

This document explains where to go depending on the kind of help you need.

## Before Opening an Issue

Please check these first:

- [README.md](README.md)
- [docs/index.md](docs/index.md)
- [docs/user-guide.md](docs/user-guide.md)
- [docs/config-reference.md](docs/config-reference.md)
- [docs/troubleshooting.md](docs/troubleshooting.md)

Many questions about installation, profile setup, query syntax, and output
targets are already covered there.

## Use the Right Channel

### Bug Reports

Open a bug report issue when:

- something is broken
- behavior differs from the docs
- you can provide steps to reproduce

Include:

- what you expected
- what happened instead
- the Obsidian version
- the Promptfire version
- relevant settings or vault-config context

### Feature Requests

Open a feature request when:

- you want new functionality
- you have a workflow gap the plugin should support
- you can explain the use case and expected behavior

### Documentation Improvements

Open a docs issue when:

- setup instructions are unclear
- a guide is missing a step
- wording or examples should be improved

### Usage Questions

If you are unsure whether something is a bug, start by checking the
troubleshooting and configuration docs. If it still looks like a product gap or
documentation problem, open the closest matching issue template and describe
what you were trying to do.

## What This Repository Is Best For

This repo is best for:

- product bugs
- missing or unclear documentation
- concrete feature proposals
- contributor improvements

It is not a general-purpose AI support desk. Keep reports focused on the plugin
and the exact Promptfire workflow involved.
